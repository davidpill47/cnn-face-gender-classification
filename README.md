# CNN Face Gender Classification

Deep Learning project developed during the Master's in Intelligent 
Interactive Systems at UAM (Madrid).

## About
Convolutional Neural Network to classify gender from facial images 
using the LFW (Labeled Faces in the Wild) dataset.
Two approaches were compared: custom CNN architecture vs. Transfer Learning.

## Content
- `1-CNN_clasificacion_caras_color.ipynb` — Custom CNN architecture
- `2-CNN_clasificacion_caras_color_transfer_learning.ipynb` — Transfer Learning
- `practice4.pdf` — Project description

## Technologies
Python · Keras · TensorFlow · scikit-learn · OpenCV · Jupyter Notebook
